CSS Padding

É o preenchimento do elemento, ou seja, o espaçamento interno do elemento.

Exemplo:

Esse elemento tem um espaçamento de 90px

Lados individuais

Css tem propriedades para especificar o espaçamento de cada lado de um elemento:

  • padding-bottom: indica o tamanho na parte inferior;
  • padding-left: indica o tamanho no lado esquerdo;
  • padding-right: indica o tamanho no lado direito;
  • padding-top: indica o tamanho na parte superior;

Exemplo:

Esse elemento tem um bottom padding de 20px, um padding left de 10px,
um padding right de 5px e um padding top de 7px.

CSS Margin

É a margem do elemento, ou seja, o espaçamento externo do elemento.

Exemplo:

Esse elemento tem uma margem de 90px.

O CSS tem propriedades para especificar o espaçamento de cada lado da margem.

  • margin-top: para determinar a distância superior;
  • margin-right: para determinar a distância no lado direito;
  • margin-bottom: para indicar o espaçamento inferior;
  • margin-left: para informar o espaçamento esquerdo.

Exemplo:

Esse elemento tem um margin top de 100 px, um margin bottom de 100 px,
um margin right de 150 px e um margin left de 80px.
